Workshop Overview

This workshop aims to deepen students' understanding of how economic and social crises affect communities and institutions in Jerusalem and Morocco, with Jerusalem as a central case study. We explore how digital platforms and AI techniques can support crisis analysis, institutional resilience, and the protection of cultural and architectural heritage in times of emergency.

Workshop Objectives

  1. Deepen students' understanding of how economic and social crises affect communities and institutions in Jerusalem and Morocco, with Jerusalem as a central case study.
  2. Explore how digital platforms and AI techniques can support crisis analysis, institutional resilience, and the protection of cultural and architectural heritage in times of emergency.
  3. Foster collaborative, interdisciplinary work between Moroccan and Palestinian students, and encourage critical, creative thinking about using technology in the service of justice, dignity, and memory.

Student Resources & Access

Students participating in this workshop will need to access the student repository which contains templates, resources, and instructions for developing their digital platform concepts.

Getting Started with the Student Repository

1

Access the Repository

The student repository is available at:

https://github.com/hnuseibeh/winter-university

Scan to access the repository

2

Clone or Download the Repository

Option A: Using Git (Recommended)

git clone [repository-url]
cd [repository-name]

Option B: Download as ZIP

If you're not familiar with Git, you can download the repository as a ZIP file from GitHub and extract it to your computer.

3

Quick Start Steps

  1. Read WORKSHOP_BRIEF.md to understand the workshop objectives (available in Arabic, English, and French)
  2. Navigate to the student-template/ folder
  3. Open project-concept.md and start documenting your team's idea
  4. Review resources/ folder for background context and examples
  5. Use presentation-guide.md to prepare your 3-5 minute presentation
4

What You'll Find in the Repository

  • WORKSHOP_BRIEF.md - Complete workshop instructions in 3 languages
  • QUICK_START.md - 3-step quick start guide
  • student-template/ - Your working directory
    • README.md - Detailed student instructions
    • project-concept.md - Template to document your idea
    • presentation-guide.md - How to prepare your presentation
    • resources/ - Background data and context
  • reference-dashboard/ - Example dashboard (4 pages)
  • reference-examples/ - Comprehensive example (8 pages)
